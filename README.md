《香廁聞記》私隱權政策
最後更新日期：2026-06-17

《香廁聞記》（以下稱「本應用程式」）重視你的私隱。本政策說明我們如何收集、使用及保護你的資料。

1. 我們收集的資料
我們可能收集以下資料（視你使用功能而定）：

裝置與技術資料：裝置型號、作業系統版本、裝置語言、應用程式版本、崩潰與診斷資料
遊戲進度資料：關卡進度、分數、道具與角色相關資料
廣告與分析資料：廣告成效、互動事件、匿名識別資料
帳戶/雲端同步資料（如適用）：例如第三方登入識別碼、排行榜名稱、好友代碼
2. 我們如何使用資料
我們使用資料作以下用途：

提供與維持遊戲功能（儲存進度、排行榜、好友、活動）
分析與改善遊戲體驗（平衡性、效能、錯誤修復）
防止濫用、作弊或詐騙
顯示廣告與衡量廣告成效（如有啟用）
3. 第三方服務
本應用程式可能使用第三方服務（視版本而定），例如：

Google Play 服務（如排行榜、成就、付款）
廣告服務（如 Google AdMob / Unity Ads）
分析與後端服務（如 PlayFab / Firebase）
第三方可能依其私隱政策處理資料。

4. 兒童私隱
如果本應用程式的目標對象包含未滿 13 歲兒童，我們會依適用法規及平台政策處理資料。
家長或監護人如有疑問，可透過下方聯絡方式與我們聯絡。

5. 資料保存與安全
我們會在達成上述目的所需期間內保存資料，並採取合理技術與管理措施保護資料安全。

6. 你的權利
你可依所在地法規要求查閱、更正或刪除個人資料（如適用），也可要求停止特定資料處理。

7. 政策更新
我們可能不時更新本政策。更新後將於本頁公布最新版本與日期。
